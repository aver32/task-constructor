# MVP Система тестирования стажёров

## Описание проекта

Django веб-сервис для автоматизации проверки тестовых заданий студентов на стажировку.

## Функциональность MVP

### Для организаторов:
- Создание тестов с гибкими настройками
- Добавление вопросов разных типов:
  - Одиночный выбор
  - Множественный выбор
  - Ввод текста
  - Ввод числа
  - Соотнесение (в разработке)
  - Упорядочивание (в разработке)
- Настройка проходного балла
- Просмотр статистики по тестам
- Управление через админ-панель

### Для студентов:
- Просмотр доступных тестов
- Регистрация для прохождения теста
- Прохождение теста
- Просмотр результатов с детализацией

## Установка и запуск

### 1. Создайте виртуальное окружение:
```bash
python -m venv venv
source venv/bin/activate  # Linux/Mac
# или
venv\Scripts\activate  # Windows
```

### 2. Установите зависимости:
```bash
pip install -r requirements.txt
```

### 3. Создайте файл .env в корне проекта:
```
SECRET_KEY=your-secret-key-here
DEBUG=True
```

### 4. Примените миграции:
```bash
python manage.py makemigrations
python manage.py migrate
```

### 5. Создайте суперпользователя:
```bash
python manage.py createsuperuser
```

### 6. Запустите сервер:
```bash
python manage.py runserver
```

### 7. Откройте браузер:
- Главная страница: http://127.0.0.1:8000/
- Админ-панель: http://127.0.0.1:8000/admin/

## Структура базы данных

Реализованные модели:
- **User** - пользователи (организаторы)
- **Test** - тесты
- **Question** - вопросы
- **Student** - студенты
- **Attempt** - попытки прохождения
- **Answer** - ответы студентов

## Использование

### Создание теста:
1. Войдите в админ-панель
2. Перейдите в раздел "Тесты" → "Добавить"
3. Заполните информацию о тесте
4. Добавьте вопросы через раздел "Вопросы"

### Прохождение теста студентом:
1. Студент переходит по ссылке на тест
2. Заполняет регистрационную форму
3. Проходит тест
4. Сразу видит результаты
